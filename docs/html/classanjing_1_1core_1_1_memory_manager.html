void MemoryManager::Clean ( )

Removes all allocations. This will make MemoryManager state looks like it is just newly instantiated.

Be careful to use this function. This will wipe out all allocations. We must make sure no other code has ownership of any of the allocations before performing this function.

int MemoryManager::Free ( void *  address,
const char *  filename,
unsigned int  line 
)

Deallocate memory. Returns 1 if address is successfully deallocated.

if address is nullptr, the function does nothing and returns 2 if address is not allocated using Alloc(or by global operator new overrided by MemoryManager), it causes undefined behaviour this function does not change the value of address itself, hence it still points to the same (now invalid) location

Returns 3 if sentinel code in address is not intact, indicating that there is memory overrun. Address is still deallocated.
